# Nhật ký Ngày 1: Làm quen hệ thống

## 1. Cấu trúc thư mục đã học
- `/`: Gốc (Root)
- `/home`: Dữ liệu người dùng
- `/etc`: File cấu hình hệ thống
- `/var/log`: Nhật ký hệ thống

## 2. Các lệnh đã thuộc
- `pwd`: Xem đường dẫn hiện tại
- `ls -l`: Liệt kê file chi tiết
- `cd`: Di chuyển giữa các thư mục
