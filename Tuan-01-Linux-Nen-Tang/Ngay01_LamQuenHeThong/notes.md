## 1. Cấu trúc thư mục đã học

- `/`: Gốc (Root)
- `/home`: Dữ liệu người dùng
- `/etc`: File cấu hình hệ thống
- `/var/log`: Nhật ký hệ thống

## 2. Các lệnh đã thuộc

- `pwd`: Xác định vị trí hiện tại.

- `ls`:
  
  - `ls -l`: Xem chi tiết (quyền hạn, ngày tạo).
    
    <img width="809" height="546" alt="image" src="https://github.com/user-attachments/assets/2919e6a9-3a53-48a9-b661-3881fa6b85c9" />

- `ls -a`: Xem file ẩn (những file bắt đầu bằng dấu chấm `.`).

- `ls -lh`: Xem kích thước file theo đơn vị dễ đọc (KB, MB).

- `cd`:
  
  - `cd /`: Nhảy thẳng ra gốc.
  
  - `cd ..`: Lùi lại một cấp thư mục.
  
  - `cd ../../..`: Lùi lại một cấp/ hai cấp / ba cấp thư mục.
  
  - `cd ~`: Quay về thư mục cá nhân nhanh nhất.
